# Image Creator service

## Description

The **Image Creator** service is a microservice that handles the creation and management of images for various projects. It provides endpoints to upload, download, and manage image files.

## Software and Requirements

Tools/libraries: Java, Spring, Config server, Eureka Discovery, bucket4j, TwelveMonkeys ImageIO, Maven, RabbitMQ, Lombok
