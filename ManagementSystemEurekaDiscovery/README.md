# Eureka Discovery

## Description

This project is a simple Spring Boot application that acts as a Eureka Discovery Server. It allows other applications to register and discover each other using the Eureka service registry.

### Software

**Tools/libraries:** Java, Spring, Eureka Discovery, Spring actuator, Maven

### Default Configuration

- **Default spring profile:** `local`
- **Default server port:** `8761`

### Start the application

- Via terminal:
    ```bash
    mvn spring-boot:run
    ```
